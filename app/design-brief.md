# Design brief v2 — Rail Train Service (полный редизайн, плакатное направление)

Выбор направления сделан пользователем: «Плакатный цветной» (bold studio solid).
v1 (кобальт+крем, film scrub) отклонена пользователем. Сайт мультистраничный:
структура URL оригинала rtrain.ru сохраняется 1:1 (SEO), контент из content/pages.json,
галерея из content/gallery (81 реальное фото).

## Design read
Снабженцы и логисты, отправляющие грузы из Москвы в Сибирь и на Дальний Восток; регистр:
уверенный плакат советского транспортного модернизма, переосмысленный современно. Цвет главный, фото документальные.

## Concept spine
**Транспортный плакат / «Грузы едут на восток».** Каждая страница — плакат: цветное поле,
гигантская типографика, цифры как графика (км, сутки, тонны), реальные фото погрузки как
документальные вставки-«марки» на цветном поле.

## Delivery tier
cinema (Lenis+GSAP, kinetic type hero, скролл-главы).

## Locked palette — «Бутылочный плакат» (выбор пользователя)
- Field: `#0B4D3A` (глубокий бутылочный зелёный, главный цвет)
- Ground: `#F2EDE3` (крем/кость)
- Ink: `#131F1A` (тёмно-зелёные чернила)
- Accent: `#F2B02D` (сигнальная охра: жилеты, маркировка на контейнерах)
Defense: цвет выбран пользователем из превью; материальный мир: зелёные вагоны, сигнальная
маркировка, крафт-бумага накладных. Секции чередуют поля: зелёный → крем → чернила.

## Locked type
- Display: **Unbounded** 700/900 (кириллица, широкий плакатный гротеск, Google Fonts)
- Body: **Manrope** 400/500/700 (кириллица)
- Data/mono: **JetBrains Mono** (цифры, тарифы, км)
Self-hosted woff2 (cyrillic+latin).

## Tier-1 technique
**D3 — Kinetic type opener** (wow-catalog): гигантский заголовок хиро хореографируется скроллом:
строки едут по разным трекам (как составы по путям), цифра «9 288 км» проезжает под текстом;
на цветном поле с документальной фото-вставкой. Defense: спайн «плакат» = типографика и есть
изображение; движение строк = движение составов. Reduced-motion: статичная композиция.
Mobile: укороченные треки. Скриншот-безопасно: только transform, стартовое состояние собрано.

## Анти-конвергенция vs v1 (≥4 осей)
1. Палитра: кобальт/крем → бутылочный/крем/охра ✓
2. Пара шрифтов: Golos+Inter Tight → Unbounded+Manrope ✓
3. Архитектура хиро: full-bleed фото-канвас → плакатное цветное поле, typography behemoth ✓
4. Tier-1: A1 film scrub → D3 kinetic type ✓
5. Гарнитуры CTA: полностью новый набор (ниже) ✓
6. Углы: острые (сохраняются; 5/6 осей отличаются)

## Структура сайта (56 URL, сохраняется для SEO — content/pages.json)
Шаблоны:
- **home** `/` — плакатный хиро (D3) → услуги-плакаты (5 цветных полос с фото-марками) →
  маршрут с цифрами → тарифы (чернильное поле, таблица) → галерея-превью (реальные фото) →
  отзывы → контакты + форма (D1 leads, уже есть)
- **service** (~20 URL: /container, /oversized, /cars/*, …) — плакатная шапка (поле + гигантский H1
  + фото-марка) → редакционный контент из pages.json (тексты сохраняются, таблицы стилизуются) →
  соседние услуги → CTA-лента
- **city** (~20 URL: /zhd-perevozki-v-*, /zhd-otpravki-v-*) — шапка «Москва → {Город}» с км/сутками
  как графикой → текст страницы → строка тарифа (если есть в прайсе) → CTA
- **gallery** `/gallery` + 7 альбомов — плакатная сетка реальных фото, alt-подписи
- **static**: /o-kompanii, /prices, /contacts, /dokumenty, /otzivi
Title/description каждой страницы — из pages.json как есть.

## Section plan главной (7 секций, семьи не повторяются, эйбрау ≤3)
1. Hero — kinetic type poster, зелёное поле. (family: typography behemoth)
2. Услуги — вертикальный стек полноширинных цветных полос-плакатов, чередование полей, фото-марка справа. (family: poster-stacked storytelling)
3. Маршрут — крем, цифры как графика (9 288 км, 10 суток, 33 станции), линия с вехами. (family: oversized metrics strip)
4. Тарифы — чернильное поле, швейцарская таблица, mono. (family: data table) [eyebrow «Тарифы»]
5. Галерея-превью — мозаика реальных фото 2:3:2 со ссылкой в галерею. (family: gallery-led cadence) [eyebrow «Фотогалерея»]
6. Отзывы — крем, две колонки цитат без карточек, divide-y. (family: editorial quotes)
7. Контакты + форма — зелёное поле, слева реквизиты, справа форма (D1). (family: split panel) [eyebrow «Контакты»]

## CTA inventory (bespoke chrome, новый набор гарнитур)
- **«Рассчитать перевозку»**:
  - hero: билет-CTA: прямоугольник с перфорацией по левому краю (жд билет), на hover «отрывается» (сдвиг + лёгкий skew) [garment: ticket/coupon]
  - nav: штамп: :active физически «пропечатывается» (scale+rotate чуть), охра [garment: stamp/press]
  - тарифы: гигантская цифра тарифа как hit area, подпись-caption [garment: oversized numeral]
  - форма submit: mono-readout, лейбл «печатается» на hover [garment: mono decode]
- **«Отследить контейнер»**: обычная mono-ссылка в контактах → WhatsApp
Запрещённая тройка (underline draw / flood-fill / framed block): не используется вовсе.

## Copy rules
